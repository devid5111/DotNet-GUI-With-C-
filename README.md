**# DotNet-GUI-With-C-**# C# Console Programs

This repository contains C# Console Application programs developed using the .NET platform. These programs are created for learning C# fundamentals, problem-solving, and Object-Oriented Programming (OOP) concepts.

## Requirements

- .NET SDK
- Visual Studio 2022/2026 or Visual Studio Code

## How to Run

1. Open the project in Visual Studio.
2. Build the project.
3. Press **Ctrl + F5** to run without debugging or **F5** to run with debugging.

Or using the .NET CLI:

```bash
dotnet run
